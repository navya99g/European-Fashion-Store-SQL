# European Fashion Store Data Analysis using SQL

## Project Overview
This project analyses European fashion retail data to derive insights on sales performance, customer behavior, and product trends using SQL.

## Dataset
The dataset comprises customer, product, sales, and sales item details from a European fashion store.

## Objectives
- Analyse sales and revenue trends
- Identify top-performing products and regions
- Understand customer purchase behaviour

## Tools Used
- SQL
- PostgreSQL
- pgAdmin

## Key Analysis Performed
- Revenue and sales analysis using aggregations
- Customer segmentation using joins and subqueries
- Product ranking using window functions
